# PetNanny 寵物保姆

一對一的寵物保姆媒合平台，使用者可以選擇成為家長或保姆兩種角色，透過雙方約談的方式，找到最適合您的保姆

[<img width="215" height="83" src="https://firebasestorage.googleapis.com/v0/b/triptrip-5f1ab.appspot.com/o/google-play-badge.png?alt=media&token=0d0f7868-0e2c-49ab-bc1d-33e70daa6bfd"/>](https://play.google.com/store/apps/details?id=com.nicole.petnanny)

## 功能介紹
### 首頁
+ 家長可以依照服務類別選擇需要的服務 
+ 下方區域-保姆列表也提供最新的保姆讓家長做選擇

<img src="https://i.imgur.com/knpHzi2.jpg" width="190" height="360"></a>

### 保姆清單頁
+ 家長選擇服務類別後可再次更改類別
+ 還可以進一步選擇寵物類型及服務地區作為條件篩選

<img src="https://i.imgur.com/2H56S40.png" width="190" height="360"></a>

### 保姆詳細頁
+ 提供家長瀏覽保姆詳細的服務資訊
+ 家長可點擊右下方聯繫保姆，填入需求資訊，向保姆申請預約，完成後自動開啟雙方聊天室

<img src="https://i.imgur.com/UxOAsZC.png" width="190" height="360"> </a>
<img src="https://i.imgur.com/iXYH9fL.png" width="190" height="360">

### 訂單頁
+ 家長與保姆可以查看訂單詳細資訊及更改訂單狀態

<img src="https://i.imgur.com/ltxJHVs.png" width="190" height="360"> </a>
<img src="https://i.imgur.com/ZSlG4sI.png" width="190" height="360"> 
 
### 聊天室頁
+ 透過一對一聊天室做服務前的溝通確認，服務進行時保姆可以隨時回報寵物的狀態，讓家長更放心

<img src="https://i.imgur.com/PizPKWb.png" width="190" height="360"> </a>
<img src="https://i.imgur.com/iXYH9fL.png" width="190" height="360">

### 個人頁
+ 可編輯個人資訊

<img src="https://i.imgur.com/TC8xv2F.png" width="190" height="360">

+ 家長或保姆都可以新增個人心愛的毛寶貝資訊

<img src="https://i.imgur.com/CPp2LM5.png" width="190" height="360">

+ 成為保姆前必須先進行申請，填寫審查資料通過後才可以發佈服務

## 使用技術與工具

- 設計模式：MVVM、Singleton
- Android Jetpack - ViewModel、LiveData、Lifecycle、Data Binding、Navigation
- 第三方套件：Glide、ImagePicker、Lottie
- 分析工具：Firebase Crashlytics


## 開發環境
- Android Studio 4.0.1
- Android SDK 28
- Gradle 4.0.1
